# 🎓 Anurag Engineering College Website UI Redesign

## 📌 Project Overview

This project focuses on redesigning the Anurag Engineering College website to improve usability, accessibility, visual hierarchy, and mobile responsiveness.

The redesign was created as part of a UI/UX design task and follows modern design principles, heuristic evaluation findings, and accessibility guidelines.

---

## 🎯 Objectives

- Improve website usability
- Simplify navigation structure
- Enhance visual hierarchy
- Increase accessibility compliance
- Improve student-focused interactions
- Create responsive layouts for mobile and desktop

---

## 🔍 Heuristic Evaluation

The existing website was evaluated using Nielsen's 10 Usability Heuristics.

### Key Issues Identified

#### Navigation
- Too many top-level navigation items
- Lack of active navigation indicators
- Information difficult to discover

#### Privacy
- Student login not prominently visible
- Limited visibility of privacy-related information

#### Visual Hierarchy
- Multiple competing focal points
- Weak call-to-action emphasis

#### Consistency
- Inconsistent button styles
- Lack of unified design system

#### Accessibility
- Contrast improvement opportunities
- Missing accessibility enhancements

#### Error Prevention
- Limited search discoverability
- Navigation recovery improvements needed
